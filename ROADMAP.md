| Задача | Описание | Критерий готовности |

### 📚 Адаптивное обучение (IRT)
```
📐 Формула мастерства:
P(mastery) = 1 / (1 + e^-(score - δ))

Где:
- score = текущий результат пользователя
- δ = сложность концепции (калибруется на исторических данных)
```

| Функция | Описание | Источник данных |
|---------|----------|----------------|
| Динамическая сложность | Подбор вопросов по уровню владения | `user_progress` + IRT-модель |
| Персонализированные объяснения | Генерация пояснений под "слабые места" | GPT-4o + контекст ошибок |
| Рекомендации контента | Предложение тем для повторения | Анализ матрицы компетенций |
